# ai-infra

Personal AI infrastructure for tracking cognitive state across Claude Code sessions.

## Skills

- `/ai-infra:extract` — Scan current session for cognition/thinking/preference signals
- `/ai-infra:sync` — Distribute extracted signals to domain files
- `/ai-infra:setup` — Initialize infrastructure (placeholder)

## Installation

```bash
/install /path/to/ai-infra
```
